人流量统计方法可以大致分为两类：基于行人
检测的方法和基于特征回归的方法。基于行人检测的方法依赖于分割人或人头数，或基于小波
描述子、纹理分析等的行人检测方法，是通过对行人的准确定位来精确计算行人的数量(每检测到一个行人，计算的行人数量增加1)。而第二种方法是通过提取前景区域，并假设
前景中含有行人来估计人群密度，这种方法通过对一组提取的低层次特征大小的估计来表示由背景减
法或运动分割方法所假定相应的前景区域(一般可以认为提取到的都是人群信息，
然后用这一区域的特征和训练好的回归器估计这一区域大致的行人数量)，此类方法不涉及具体的行人检测，计算量小，但是无法得到精确的行人
数量，可适用于粗略的估计人群密度。

需要注意的一点是：在第二种根据密度来估计行人数量时，若是有远近景之分，
不然近景提取到的行人区域要比远景得到的行人区域大，不同的行人尺度会造成较大的估计误差。

基于检测跟踪的人数统计(适用于小场景)：
1. 目标检测：利用目标检测算法对视频中的运动人体进行检测。
2. 目标跟踪：在监测区域内设置跟踪区域(Tracking  Region)，并在该区域内
实时跟踪运动目标.
3. 计数：在跟踪过程中如果有目标跨过预先设定的计数线(Counting  Line)，
即可视为某方向上有运动目标通过，对通过计数线的目标进行计数操作.

人群密度分析(适用于大场景)：
1. 提取前景：图像序列中将变化区域从背景图像中提取出来。
2. 透视分析：处理人体在图像中的大小随着离摄像头距离的远近而造成的不同。
3. 人数统计：根据提取到的前景拟合或回归法得到人群数量。

红外行人检测与跟踪：
由于红外探测是利用物体的温度反应成像，因此能够不受光照的影响，也因此特殊材料的衣服或是冬天衣着较厚的情况下，
会使得接受到的人体温度和背景无法区分
